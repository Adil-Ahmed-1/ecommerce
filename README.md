# 🛒 E-Commerce Web Application (Frontend + Backend)

A full-stack E-commerce web application built with modern web technologies. This project includes both **frontend UI** and **backend APIs** with database integration for managing products, categories, cart, and users.

---

## 🚀 Features

### 👨‍💻 Frontend
- Responsive modern UI
- Product listing with categories
- Product detail page
- Shopping cart system
- Search and filter products
- Mobile-friendly design

### ⚙️ Backend
- RESTful APIs
- User authentication (Login/Register)
- Product management (CRUD)
- Category management
- Cart & Order handling
- Secure database connection
